# pomodoro.sh
A dumb pomodoro bash program

## Installation

```bash
git clone https://github.com/Em-ilien/pomodoro.sh.git
cd pomodoro.sh
chmod u+rx .
```

# Usage

```bash
# Start
./pomodoro.sh &

# Stop
pkill pomodoro.sh
```
